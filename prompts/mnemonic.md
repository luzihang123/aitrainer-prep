# Prompt: 背诵口诀（mnemonic）

## 任务

为给定题目生成背诵口诀——考前 7 天的"突击版"内容，帮考生快速记忆核心要点。

## 风格要求

- **极短**：核心口诀控制在 20 字以内
- **可朗朗上口**：押韵、对仗、谐音都欢迎
- 配 1-2 句最简记忆要点
- 不解释原理（那是 explanation 的事）

## 好坏对比

题目：1.2.x 准确性不高的对策

- ✅ 好：
  ```
  口诀：准确不高，数据 + 算法

  - 数据三件套：清洗 / 扩充 / 标注
  - 算法两件套：升级 / 调参
  ```

- ❌ 差：
  ```
  对于准确性不高的问题，应该从数据质量和模型算法两个方面进行优化，
  数据质量优化包括数据清洗、数据扩充和数据标注，模型算法优化包括
  升级模型和调整参数。
  ```
  （问题：太长、像复述课文，没有突击复习的"卡片感"）

## 适用场景

- ✅ 适用：套路化的理论题、API 列表、对应关系
- ✅ 适用：同类知识点的对比记忆
- ⚠️ 谨慎：复杂概念题——这种题强行编口诀反而误导
- ❌ 不用：单一定义题——直接背原文就行，不需要口诀

如果题目不适合编口诀，输出 `（本题不适合背诵口诀，建议直接看 explanation）` 即可。

## 输出格式

直接输出 markdown 正文，可包含代码块用于格式化口诀。

## 输入

题目的 frontmatter（含 `explanation` 已生成则更好）。
