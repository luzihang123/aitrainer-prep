# 版本切换

不同城市、考点或培训体系下，报名政策、考试组织、题型细节和补贴规则可能不同。本站用独立路径区分版本，避免把不同来源的信息混在一起。

## 当前版本

| 版本 | 路径 | 状态 | 说明 |
|------|------|------|------|
| 上海版 | [查看](/shanghai/) | 建设中 | 当前站点内容主要基于上海考试经验整理 |
| 深圳版 | [查看](/shenzhen/) | 待补充 | 等待深圳考生补充真实考试与政策信息 |

## 路径规则

- 城市版本：`/shanghai/`、`/shenzhen/`
- 考点或学校版本：后续可用 `/sjtu/`、`/computer-postgrad/` 这类短路径
- 通用内容仍放在 `/intro/`、`/modules/`、`/cram/`、`/graph/`

## 维护原则

- 通用考点放在公共模块里，避免重复维护。
- 城市政策、考点体验、报名流程、补贴信息放在对应版本页。
- 没有真实来源的信息先标记为待补充，不用 AI 硬写。
