## 三个最重要的认知

如果时间紧只读三句话，请记住：

1. **70 分都在 Python 编程**——这是绝对重点
2. **题目不提供代码框架，只需背诵 import 语句和核心 API 调用**——这是颠覆很多人备考方式的关键技巧
3. **第三部分 Excel（15 分）是最高性价比内容**——通常套路化、操作有限、最容易拿满分
