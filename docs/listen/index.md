# 四、听题模式 🎧

> 通勤、洗碗、睡前都能学。把考试装进耳朵里。

## 状态

🚧 听题播放器组件正在开发中。预计 Phase 1 上线。

## 设计理念

听题模式的用户**手是不空的**——通勤、洗碗、健身、睡前。所以一切设计为"不看屏幕"服务。

## 默认朗读节奏

```
读题目 → 停 3 秒（思考）→ 读"答案是 X" → 读一句话解释 → 下一题
```

## 计划功能

- [ ] 全部题库播放
- [ ] 按部分听（一 / 二 / 三 / 四）
- [ ] 按场景听（数据预处理 / 建模 / 图像识别 / 理论）
- [ ] 错题循环
- [ ] 高频考点精选（30 分钟版）
- [ ] 速度控制（0.8x / 1.0x / 1.25x / 1.5x）
- [ ] 思考停顿时长可调
- [ ] 后台播放（锁屏可控）
- [ ] 耳机线控（双击下一题）

技术实现见 [产品设计文档第 5.2 节](/about/design)。
