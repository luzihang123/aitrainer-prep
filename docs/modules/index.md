# 二、考点详解

> 系统化讲解每个考点。每道题包含原理图解、出题人视角、排除法、知识图谱关联。

## 学习路径建议

按性价比顺序：

1. **[通用考试策略](./imports)** —— 必读，所有 Python 题的基础
2. **⭐ [Excel 部分](./excel/)（15 分）** —— 优先吃透
3. **[Python 数据预处理](./python-prep/)（35 分）** —— 重点投入
4. **[Python 建模](./python-model/)（20 分）** —— 套路化
5. **[Python 图像识别](./python-image/)（20 分）** —— 流程化
6. **[理论部分](./theory/)（30 分）** —— 模板化

## 每个知识点的页面结构

每个知识点都按统一模板组织：

- **这是什么**：一句话定义
- **为什么重要**：业务/工作意义
- **实际场景**：真实例子
- **核心 API 与背诵口诀**
- **原理图解**
- **真题精讲**（含出题人视角、排除法、知识图谱）
- **背诵卡片**（精简版）
- **易错点**
