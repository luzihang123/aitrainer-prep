# 关于本站

## 项目愿景

让备考从"背题"变成"懂题"，从"必须坐下看"变成"随时能听"。

做一个**开源、免费、原理优先、AI 辅助**的备考站，建立人工智能训练师备考方向的事实标准内容库。

## 站长是谁

一位 2026 年通过人工智能训练师三级（高级）考试的考生。考完后发现资料零散、培训机构靠"背题"，决定把备考过程系统化、开源化。
